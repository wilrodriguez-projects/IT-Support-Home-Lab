# Lab 01 – Windows Virtual Machine Deployment

## Objective

Deploy a Windows virtual machine to simulate a help desk troubleshooting environment.

This lab is part of my IT Support Home Lab while preparing for the CompTIA A+ certification.

---

## Lab Environment

Host Machine  
Surface Laptop 5  
Intel i7 12th Gen  
16 GB RAM  
Windows 11

Virtualization Platform
Hyper-V

Guest OS  
Windows 10 / Windows 11

---

## Tasks Performed

• Installed virtualization platform  
• Created a Windows virtual machine  
• Allocated CPU, memory, and disk resources  
• Mounted OS installation media  
• Installed Windows OS inside VM

## Skills Practiced

• Virtualization fundamentals  
• Operating system installation  
• Resource allocation  
• System configuration
